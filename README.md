📌 Smart Task Manager
🚀 Overview

Smart Task Manager is a multi-user, secure productivity web application built using Python and Streamlit. It allows users to efficiently manage daily tasks, track productivity, and visualize insights through an interactive dashboard.
Each user has a private workspace with authentication (Signup/Login), ensuring complete data isolation and security.

✨ Features:

🔐 Authentication System:

Secure Signup and Login system
Password encryption using SHA-256 hashing
Session-based user authentication
Private task storage per user

📋 Task Management:

Add, view, and manage daily tasks
Set task priority (Low / Medium / High)
Mark tasks as Completed or Pending
Track task creation time and deadlines
Download tasks as CSV file

📅 Unified Task Dashboard:

Combined Calendar + Task List View
Real-time filtering by:
Status (Pending / Completed)
Priority (Low / Medium / High)
Clean and structured table view
Interactive task completion updates

📊 Analytics Dashboard:

Pie chart: Task completion overview
Bar chart: Priority distribution
Productivity score based on completion rate
Visual insights for performance tracking

🎨 UI/UX Design:

Modern dark green SaaS-style interface
Responsive sidebar navigation
Clean and minimal dashboard layout
Smooth user experience with Streamlit components

🛠️ Tech Stack:

Frontend: Streamlit
Backend: Python
Data Handling: Pandas
Visualization: Matplotlib
Security: Hashlib (SHA-256)
